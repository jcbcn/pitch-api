# Card API

| Resource               	| POST                                	| GET                             	| PUT 	| DELETE 	|
|------------------------	|-------------------------------------	|---------------------------------	|-----	|--------	|
| /card?{skip}&{take}&{position}&{notIn}       	|                                     	| Retrieve all cards           	|     	|        	|
| /card/{id}              	|                                     	| Retrieve card by id            	|     	|        	|
| /card/cards/{id1;id2;id3...}       	|                                     	| Retrieve cards by id 	|     	|        	|
